Chat App with PDF Reader

Chat App with PDF Reader, This App is a pdf book reading as well as a chat app also. Book lovers of our country who love to read book in their smartphone, they are the targated client of this App. Without Login/Registration anyone can read the book.But If anyone want to chat with other book lovers who has also completed their registration in this app, will be able to chat with each other. Here is the github link of this App_ https://github.com/Karibayasmin/Chat-App-with-PDF-Reader.git

App Features

  • Any time without Login/Registration any Book lover can read his/her favourite book in this app using their Android Phones.
  • If anyone want to find other Book Lover, in this platform and want to chat with them, they can do by completed Login/Registration process.

USED TECHNOLOGIES/LIBRARIES

  • In Home page Preferable book option and Login option available. Preferable book option is just a Button widget. and Login option is designed with Edit Text widget.
  • Login/ Registration option is connected with Firebase realtime database which is free for some limited space.
  • And For PDF Reader, I have used an library from github which has made my work easy

My Role

  • I was the Developer of this App.
